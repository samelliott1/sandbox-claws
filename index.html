Setup Guide

1

Mac Mini VM Setup

# Recommended VM configuration
VM Type: Ubuntu 22.04 LTS or Debian 12
CPU Cores: 2-4 (limited)
RAM: 4-8 GB (limited)
Disk: 50 GB (isolated volume)
Network: NAT with firewall rules

Hypervisor Options: UTM (recommended for Mac), Parallels, VirtualBox, or VMware Fusion

2

Test Account Setup

  • Create new Gmail: openclaw.test.{random}@gmail.com
  • Enable Google Calendar API access
  • Set up Gmail filters for test emails
  • Configure OAuth scopes (minimal permissions)
3

GitHub Repository Setup

# Clone OpenClaw
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# Create your testing branch
git checkout -b testing/evaluation-2026

# Create documentation repo for findings
mkdir openclaw-evaluation
cd openclaw-evaluation
git init
echo "# OpenClaw Security Evaluation" > README.md
git add README.md
git commit -m "Initial commit"
4

Install OpenClaw

# Inside VM - Install dependencies
pip install -r requirements.txt

# Configure with test credentials
cp .env.example .env
# Edit .env with test account details

# Verify installation
python -m openclaw --version
